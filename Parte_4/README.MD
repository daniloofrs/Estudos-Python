Calculador de Descontos Universitário
Sistema em Python para automatizar o cálculo de mensalidades com base no desempenho acadêmico e assiduidade.

Funcionalidades
Coleta de dados do aluno (Nome, Nota e Faltas).

Validação de limites (Notas entre 0 e 10).

Cálculo automático de faixas de desconto:

30% de desconto: Nota >= 9.0 e menos de 5 faltas.

10% de desconto: Nota >= 7.0 ou baixa taxa de faltas.

Interface de terminal limpa com limpeza de tela automática.

Relatório final com lista de alunos premiados (Desconto de 30%).
